# DataAnalytics
데이터분석 프로젝트 모음집

- 미니프로젝트_1차: AIVLE에서 진행한 데이터 분석 (EDA) 주제를 바탕으로 유사 도메인 내 개인 프로젝트 별도로 진행한 내용을 업로드하였습니다.
  - AIVLE에서 진행한 미니프로젝트 1차 내용은 pdf 파일로 정리하였습니다.
  - 개인 프로젝트로 재구성한 프로젝트 중 코드 파일은 ipynb 파일로 업로드하였습니다.
  - 분석 내용을 바탕으로 한 인사이트 도출 및 정책 제안은 pdf 파일로 추가 업로드할 예정입니다.

- 미니프로젝트_2차: AIVLE에서 진행한 머신러닝(비지도학습 - 군집화) 프로젝트 내용을 정리하여 업로드 하였습니다.

- 미니프로젝트_3차: AIVLE에서 진행한 AI Agent 구축 프로젝트(팀) 진행한 내용을 업로드하였습니다.
