- 2차 미니프로젝트: 머신러닝(비지도학습 - 군집화) -> 머신러닝(혹은 딥러닝)을 주제로 도메인을 별도로 선정하여 개인 프로젝트로 별도 진행할 예정입니다.
  <br> 팀으로 진행한 프로젝트는 'DX_10반_19조' 파일 이름으로 업로드하였습니다.

- 3차 미니프로젝트: AI Agent 구축 프로젝트로 팀 프로젝트로 진행한 내용을 업로드하였습니다.
